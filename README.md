# Description
A class page with three tabs: home, stocks, and dogs. The stocks page displays stock prices given a ticker and time period, as well as how the top five stocks are doing. A bull or bear icon represents the sentiment. The dogs page contains a carousel for ten random dog images. Clicking on the buttons will display information about the listed dog breed. The option to explore the site using audio is also available, and instructions for this are written on the bottom of each page.

Note: Please disable CORS if you want the quote to load.
